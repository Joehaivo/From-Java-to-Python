## 基础

*  Hello World

```java
System.out.printf("Hello World");
```
```python
print 'Hello World'
```
*  变量

```java
int a = 233;
a = "Hello";   //错误，类型不同
```
```python
a = 233
a = "Hello"  #正确，类型可以动态改变
```
* 字符串
```java
String s = "Hello";
String s2 = "Jack";
String s3 = s + s2;
```
```python
s = 'Hello'
s2 = 'Jack'
s3 = s + s2
```
